# Overview 
This project is the Library Management System's initial phase, which focuses on application logic. This project concentrates on the logic and functionalities without implementing a database or graphical user interface. The books were stored in a Hash Map instead of a SQL database. The system's objectives include enabling users to remove books based on their barcodes and titles, checking in and out books, and displaying the library book list. Service Classes were also used in this project to manage different operations such as library collection management (add/ remove book) and book availability status (check out/ check in). 

# Features

1. Add Book from Text File
2. Remove Book By Barcode
3. Remove Book by Title
4. Check out Book
5. Check In Book 
6. Display Library List

# How to Run the Project

1. Clone Repository
2. Use Java IDE 
3. Create a text file with the title "books.txt"
- With the format (Barcode, Book Title, Author)
Example: 001, Atomic Habits, James Clear
5. Run the program using the Main class
6. Interact with the System
